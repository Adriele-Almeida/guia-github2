Passo 2: Criar Conta no GitHub
‚ñº
Siga estes passos:
  1. Acesse github.com
  2. Clique em "Sign up" (Cadastrar)
  3. Preencha seu email, uma senha forte e um nome de usuario
  4. Confirme seu email (verifique sua caixa de entrada)
Dica:

Escolha um nome de usuario profissional. Exemplo: se seu nome e Maria Silva, use algo como "mariasilva" ou "maria-silva-dev".

Passo 3: Criar Repositorio
‚ñº
Criando seu repositorio:
  1. No GitHub, clique no botao verde "New"
  2. No campo "Repository name", digite: nome-do-projeto

    Exemplo: se seu projeto e um portfolio, digite:

    meu-portfolio

  3. Marque a opcao "Public"
  4. Marque "Add a README file"
  5. Clique em "Create repository"
Muito importante:

Para sites de projeto (nao seu site pessoal), o nome pode ser qualquer um, como nome-do-projeto. Isso fara seu site ficar em: seu-usuario.github.io/nome-do-projeto/

Passo 4: Fazer Upload do Arquivo
‚ñº
Enviando seu arquivo:
  1. Dentro do repositorio que voce criou, clique em "Add file"
  2. Clique em "Upload files"
  3. Arraste o arquivo index.html que voce salvou OU clique em "choose your files"
  4. Na parte de baixo, escreva uma mensagem. Exemplo: "Meu primeiro site"
  5. Clique no botao verde "Commit changes"
Aguarde alguns minutos...

O GitHub precisa de 2 a 5 minutos para processar e publicar seu site. Depois desse tempo, seu site estara no ar!
